BLUFF Charts
A light weight charting tool...

hillbillysolutions.bluff.example.js is a hard coded example and does not query any SharePoint list or library

hillbillysolutions.bluff.js queries a standard Task list and genereates various graphs for task count by status and person assigned.
The following information is assumed:
* Task list name: "Tasks"
* Task status options:  "Completed", "Not Started", "In Progress", "Canceled"
