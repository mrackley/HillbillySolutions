The scripts in this repository use datatables and REST to create list views for lists that have well over 5,000 items in them. 

This script assumes the following"
  List containing zipcodes:  "MasterZipCode"
  Fields within the list (internal file name): "Id,Title,zip,primary_city,state,county,area_codes,timezone,latitude,longitude,estimated_population"
  An indexed column of "state"
